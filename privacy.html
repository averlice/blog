Blindsoft discord server privacy statement

We do not use a double counter, nor do we use any system that logs IP addresses for verification purposes.

Our verification bot of choice is wick which uses captcha verification not IP/user ID verification.

Why i made this post:

This is a response to doogle which  is part of the double counter.

Doogle is a service that allows for cross search across discord to “help discord moderators with detecting alt accounts”.

However, blindsoft will not use double counter.

Blindsoft did used to use double counter, though that system was flawed, and admittedly had errors that were avoidable.

The wick verification system does not use your IP address to verify you, in stead, it uses captcha verification.

While it does require your discord account for you to actually start the captcha (because it won’t know who to verify) it  will not use your IP.

So…what do we use?

Wick has something called alting protection, which uses age verification at the account level. Blindsoft does not disclose the account age you need to be in order to get verified.

However, this is one of the factors blindsoft uses in order to verify alts.

We know account age has some false positives,  but it is massively better than IP address verification via website like double counter.

Note: websites themselves may log your IP for security reasons, which is common in reverse proxies and web servers. That is on them. But for that, you should be using a vpn service or tor to hide your IP and protect yourself from IP leakage.overall conclusion

Overall conclusion

Blindsoft has not and will not use doogle, and it will no longer use double counter, so you do not have to worry about us searching through your user ID across discord. It is recommended you opt out of other servers using double counter by typing /privacy in a server that does use double counter and/or if your server uses double counter.
